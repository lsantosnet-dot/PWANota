# Sugestões de Features - AINote

Lista de ideias para evolução do aplicativo, divididas por categorias de impacto e funcionalidade.

## 1. Inteligência Artificial (O "AI" do AINote)
*   **Integração Direta com Gemini API:** Adicionar um botão para "Processar com IA" dentro do app usando a SDK do Google, salvando a resposta diretamente.
*   **Auto-Tagging:** Sugestão automática de tags baseada no conteúdo da nota via IA.
*   **Gerador de Títulos Mágico:** Gerar títulos automáticos para notas baseados no contexto do texto.
*   **Resumo Automático:** Criação de um campo "TL;DR" (resumo) para notas longas.

## 2. Organização e Produtividade
*   **Sistema de Variáveis (Templates):** Suporte a placeholders como `{{contexto}}` ou `{{idioma}}` que solicitam preenchimento ao copiar o prompt.
*   **Markdown Preview:** Renderização de Markdown nos cards para melhor visualização de listas, negritos e blocos de código.
*   **Favoritos e Fixados:** Funcionalidade de "Pin" para manter prompts importantes no topo.

## 3. Experiência do Usuário (UX/PWA)
*   **Voice-to-Prompt:** Uso da Web Speech API para ditar notas por voz.
*   **OCR (Texto de Imagens):** Integração com `Tesseract.js` para extrair texto de fotos capturadas pela câmera.
*   **Backup e Sincronização:** Opção de Exportar/Importar JSON ou sincronização com serviços de nuvem (Google Drive/Dropbox).

## 4. Estética e Personalização
*   **Dark/Light Mode Toggle:** Controle manual de tema (Claro/Escuro).
*   **Visualização em Lista vs. Grid:** Alternância entre layouts para melhor navegação em grandes volumes de notas.
